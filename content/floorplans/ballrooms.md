### The PDF viewer above allows you to:

- Navigate between pages using the Previous/Next buttons
- Zoom in and out using the +/- buttons
- Enter a specific page number directly
- **Add annotations using Hypothesis** - click anywhere on the PDF to add
  comments, highlight text, or make notes

### Here's how to use it:

1. **Add annotations**: Click anywhere on the document to add a comment
2. **Reply to annotations**: Respond to existing annotations
3. **Filter annotations**: Use the sidebar to filter and search annotations
4. **Toggle sidebar**: Click "Annotations" button to toggle sidebar

_The annotations can be accessed by anyone with the link to this document._
